Setup Guide

1 Open Settings
In your CRM account, go to Settings (gear icon in the bottom left sidebar).
2 Find Private Integrations
Navigate to Settings > Integrations > Private Integrations.

If you do not see it, enable it under Settings > Labs first.
3 Create New Integration
Click Create new Integration. Give it a name like "Profile Saver" and an optional description.
4 Set Permissions
Select these required scopes:
  • Contacts > Read/Write
  • Custom Fields > Read/Write
5 Copy the Token
After creating, you will see a token. Copy it immediately - you will not be able to see it again later.

Paste it in the extension settings under "Private Integration Token".
6 Get Your Location ID
Go to Settings > Business Info.

Your Location ID is shown at the top of the page, or in the URL bar: app.gohighlevel.com/v2/location/YOUR_ID/...
Important
Old "Location API Keys" (from Settings > Business Info > API Key) no longer work. They were deprecated in Dec 2025. You must use a Private Integration Token as described above.
